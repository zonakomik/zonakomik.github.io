6. Asa 6 ji han ni gakkou e ikimasu


Asa 6 ji han ni gakkou e ikimasu artinya adalah (aku) pergi ke sekolah jam 6.30 pagi.

Pembahasan

Mari dibahas mengenai setiap kata-perkatanya beserta juga dengan menyertakan penjelasannya :

朝 • あさ (asa)

Yaitu artinya yakni pagi, dini hari, shubuh, fajar. Kata asa masuk kedalam kata kata benda atau meishi, penggunaan kata 朝 • あさ (asa) dimulai dari waktu awal matahari terbit sampai batas waktu siang.

6時 (roku ji)

Artinya yaitu jam 6 atau pukul 6. Dimana bentuk kanji 時 (ji) yang berarti pukul atau jam, ketika ingin menyebutkan jam dengan menggunakan bahasa Jepang, maka gunakan kata 時 (ji). Pola untuk menyebutkan jam adalah berikut ini :

Angka/bilangan + 時 (ji). except; khusus untuk jam 4, 7, dan 9 terdapat penyebutannya sendiri.

1時 (ichi ji) : jam 1.2時 (ni ji) : jam 2.3時 (san ji) : jam 3.4時 (yo ji) : jam 4.5時 (go ji) : jam 5.6時 (roku ji) : jam 6.7時 (shichi ji) : jam 7.8時 (hachi ji) : jam 8.9時 (ku ji) : jam 9.10時 (juu ji) : jam 10.11時 (juu ichi ji) : jam 11.12時 (juu ni ichi) : jam 12.

半 (han)

Kalau dalam bahasa Indonesia adalah setengah (untuk menyebutkan jam yang sudah memasuki 30 menit atau lebih dari 30 menit).

Sehingga, jika digabungkan dengan 6時 (roku ji) + 半 (han) maka menjadi 6時半 (roku ji han) artinya yaitu pukul 6.30 atau jam setengah 7.

に (ni)

Tidak memiliki artiannya sendiri, karena untuk menerangkan atau menyatakan waktu dari kalimat itu, itulah salah satu fungsi dari partikel に (ni).

学校 • がっこう (gakkou)

Adalah sekolah (tempat belajar), pengucapannya termasuk bentuk umum sehingga mencakup luas (sekolah apa saja).

へ行きます • へいきます (e ikimasu)

Dapat diartikan pergi, karena adanya salah satu partikel di awal yaitu partikel へ (e) untuk menyatakan tujuan tempat yang dituju dengan jelas kemana yang akan perginya.

Kata kerja ikimasu yaitu kata kerja l. Sebelum adanya penambahan kata masu diakhir, kata kerja ikimasu awalnya adalah 行く • いく (iku).

Tujuan adanya ます (masu) di akhir -› penghalus kata kerja/kalimat.

---------

Jika setiap kata-katanya digabungkan, maka akan membentuk suatu kalimat dalam bahasa yaitu adalah : 朝6時半に学校へ行きます (asa 6 (roku) ji han ni gakkou e ikimasu).

Asa 6-ji han ni gakkou e ikimasu artinya adalah "(Saya) pergi ke sekolah pada jam 6:30 pagi".

• • • • • • • • • • • • • • • • • • • • • • • • • • • • • •

Pembahasan :

Asa「朝・あさ」

Artinya adalah "Pagi".

6-ji han「6時半・６じはん」

Dibaca 'Roku-ji han' dan berarti "Jam 6:30" atau "Jam setengah 7". Kata 'Ji' sendiri berarti "Jam", sedangkan 'Han' berarti "Setengah". Kata 'Han' bisa juga diganti dengan 'San juppun' yang artinya "30 menit".

Ni「に」

Tidak ada arti khusus, tapi pada konteks kalimat ini bisa diartikan dengan "Pada". Kata ini merupakan partikel yang salah satu fungsinya adalah untuk menunjukkan waktu.

Gakkou「学校・がっこう」

Artinya "Sekolah". Sekolah yang dimaksud di sini adalah sekolah pada umumnya, seperti SD, SMP, SMA, dll.

E「へ」

Sering diartikan dengan "Ke" atau "Untuk". Kata ini merupakan partikel yang fungsinya adalah untuk menunjukkan arah tujuan atau orang yang dituju. Penulisannya dengan huruf He, tapi dibaca E.

Ikimasu「行きます・いきます」

Artinya "Pergi". Kata 'Ikimasu' merupakan bentuk Masu dari kata kerja 'Iku'. Bentuk Masu adalah bentuk sopan, berfungsi untuk membuat katanya menjadi lebih sopan dan halus.

26. apa arti watashi no gakkou


Watashi no gakkou yang berarti adalah sekolah aku.

Pembahasan

私の学校

atau

わたしのがっこう

Watashi no gakkou

Sekolah aku/saya.

________

Watashi「 私 • わたし 」

Yaitu artinya ialah aku, saya. Penggunaanya adalah untuk menyatakan diri sendiri (kata ganti tunggal pertama), selain itu kata tersebut termasuk bentuk yang bisa dilakukan dalam situasi yang formal bahkan juga pada saat semi-formal.

No「 の 」

Penggunaannya seperti kata bantu yang bertujuan untuk menunjukkan mengenai kepemilikan, seperti yang terdapat dalam kalimat atas 'watashi no gakkou' yaitu 'sekolah (punya/milik) aku'.

Contoh lainnya seperti :

これは私のペンです (kore wa watashi no pen desu) : ini pulpenku.彼女は私の母です (kano kojo wa watashi no haha desu) : dia adalah ibuku.五条は私の夫です (Gojo wa watashi no otto desu) : Gojo adalah suami saya.

Gakkou「 学校 • がっこう 」

Yang berarti sekolah. Kata gakkou merujuk ke artian sekolah yang umum (sekolah apapun).

Watashi no gakkou artinya "Sekolahku"

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Watashi「私・わたし」

Artinya "Saya" atau "Aku". Merupakan salah satu kata ganti orang pertama tunggal dalam Bahasa Jepang. Kata ini termasuk sopan, dan bisa digunakan di situasi formal maupun informal. Kata bisa digunakan oleh perempuan maupun laki-laki.

- - -

No「の」

Biasa tidak diartikan, tapi artinya kurang lebih seperti "Milik", "Punya", dsb. Kata ini merupakan partikel atau kata bantu kalimat, yang fungsinya untuk menyatakan suatu kepemilikan. Misalkan pada kalimat 'A no B', maka dalam Bahasa Indonesia artinya dibalik menjadi "B milik A" atau "B A".

Contoh-contoh kalimat :

Anata no namae = Nama (milik) kamuKi no ue = Atas (nya) pohonAo no warutsu = Waltz biru

- - -

Gakkou「学校・がっこう」

Artinya "Sekolah". Sekolah di sini adalah sekolah pada umumnya, seperti SD, SMP, SMA, dsb.

37. apakah perbedaan part, season, cour, dan chapter dlm suatu film/komik ?​


Berikut ini penjelasan mengenai perbedaan part, season, cour, serta chapter yang biasa dijumpai pada film, buku atau komik:

Part atau bagian dalam film atau komik merujuk pada subdivisi utama yang membagi cerita menjadi bagian-bagian yang lebih kecil. Contohnya, sebuah film dengan tiga bagian yang berbeda dapat disebut sebagai "trilogi". Bagian-bagian tersebut sering kali memiliki alur cerita dan fokus yang berbeda.Season atau musim dalam konteks televisi atau anime merujuk pada serangkaian episode yang ditayangkan dalam periode waktu tertentu, biasanya dalam per sekian bulan atau bahkan satu tahun. Biasanya satu musim memiliki 12 hingga 24 episode, tergantung pada genre dan durasi episode.Cour atau kouru dalam anime adalah istilah yang digunakan untuk menggambarkan pembagian episode dalam satu musim. Istilah "cour"  awalnya digunakan menggambarkan rentang episode selama satu musim. Setiap kouru biasanya terdiri atas 10 - 14 episodeChapter adalah nama lain 'bab' yang biasa kita jmpai dalam komik atau buku. Chapter merujuk pada bagian-bagian besar cerita yang terpisah satu sama lain. Setiap chapter biasanya memiliki memiliki kisah dan alur cerita tersendiri meskipun tentu masih berhubungan dengan cerita utama.Pembahasan

Secara umum bisa dikatakan bahwa part, season, dan cour mengacu pada pembagian cerita pada media audiovisual. Adapun chapter, lebih cenderung mengacu pada pembagian cerita dalam bentuk media cetak seperti komik dan buku. Meski begitu, beberapa anime juga menggunakan istilah chapter untuk mengacu pada episode dalam serinya.

38. Douyatte gakkou e ikimasuka


Douyatte gakkou e ikimasuka? artinya dalam bahasa Indonesia berarti bagaimana caranya kamu pergi ke sekolah?

Pembahasan

どうやって学校へいきますか?

Douyatte gakkou e ikimasuka?

Bagaimana caranya kamu pergi ke sekolah?Kamu pergi ke sekolah naik apa?

______________

どうやって (douyatte)

Yang berarti artinya yaitu bagaimana caranya. merupakan bentuk salah satu kalimat tanya yang menanyakan mengenai caranya melakukan sesuatu yang ditanyakan.

学校 • がっこう (gakkou)

Artinya ialah sekolah, penggunaan kata 学校 (gakkou) tersebut termasuk penggunaan yang umum dalam menyebutkan sekolah.

へいきますか (e ikimasuka?)

Terdapat salah satu partikel yaitu へ (e) yang dimana partikel berguna untuk menanyakan ataupun menyatakan arah yang dituju tersebut.

Arti いきます (ikimasu) itu adalah pergi, bentuk kata tersebut awalnya berasal dari kata いく(iku) dan termasuk golongan l kata kerja. Karena terdapat akhiran ます (masu), sehingga kata tersebut merupakan jenis penggunaan yang formal.

Partikel か (ka) disini membantu atau berguna menjadi bentuk kata tanya.

'Douyatte gakkou e ikimasuka' artinya dalam Bahasa Indonesia adalah "Bagaimana cara (kamu) pergi ke sekolah?".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

どうやって学校へ行きますか？

どうやってがっこうへいきますか？

Douyatte gakkou e ikimasuka?

Bagaimana cara (kamu) pergi ke sekolah?

- - - - - - -

Berikut pembahasan per-kata nya.

DOUYATTE「どうやって」

Artinya "Bagaimana (cara)".

GAKKOU「学校・がっこう」

Artinya "Sekolah" pada umumnya.

E「へ」

Artinya "Ke" dan "Untuk". Kata ini merupakan sebuah partikel atau kata bantu kalimat, yang fungsinya untuk menunjukkan arah. Partikel ini ditulis dengan huruf Hiragana へ (He), tapi bacanya E.

IKIMASU「行きます・いきます」

Artinya "Pergi". Kata ini merupakan hasil konjugasi bentuk Masu dari kata kerja 'Iku' yang artinya juga "Pergi". Fungsinya bentuk Masu ini adalah untuk memperhalus dan mempersopan kata nya.

KA「か」

Tidak ada arti khusus, tapi bisa saja diartikan "...kah?". Kata ini merupakan partikel, yang fungsinya untuk membuat kalimat tanya. Biasa diletakkan di akhir kalimat setelah 'Desu' atau 'Masu'.
