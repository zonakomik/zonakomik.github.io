8. apa arti watashi no gakkou


Watashi no gakkou yang berarti adalah sekolah aku.

Pembahasan

私の学校

atau

わたしのがっこう

Watashi no gakkou

Sekolah aku/saya.

________

Watashi「 私 • わたし 」

Yaitu artinya ialah aku, saya. Penggunaanya adalah untuk menyatakan diri sendiri (kata ganti tunggal pertama), selain itu kata tersebut termasuk bentuk yang bisa dilakukan dalam situasi yang formal bahkan juga pada saat semi-formal.

No「 の 」

Penggunaannya seperti kata bantu yang bertujuan untuk menunjukkan mengenai kepemilikan, seperti yang terdapat dalam kalimat atas 'watashi no gakkou' yaitu 'sekolah (punya/milik) aku'.

Contoh lainnya seperti :

これは私のペンです (kore wa watashi no pen desu) : ini pulpenku.彼女は私の母です (kano kojo wa watashi no haha desu) : dia adalah ibuku.五条は私の夫です (Gojo wa watashi no otto desu) : Gojo adalah suami saya.

Gakkou「 学校 • がっこう 」

Yang berarti sekolah. Kata gakkou merujuk ke artian sekolah yang umum (sekolah apapun).

Watashi no gakkou artinya "Sekolahku"

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Watashi「私・わたし」

Artinya "Saya" atau "Aku". Merupakan salah satu kata ganti orang pertama tunggal dalam Bahasa Jepang. Kata ini termasuk sopan, dan bisa digunakan di situasi formal maupun informal. Kata bisa digunakan oleh perempuan maupun laki-laki.

- - -

No「の」

Biasa tidak diartikan, tapi artinya kurang lebih seperti "Milik", "Punya", dsb. Kata ini merupakan partikel atau kata bantu kalimat, yang fungsinya untuk menyatakan suatu kepemilikan. Misalkan pada kalimat 'A no B', maka dalam Bahasa Indonesia artinya dibalik menjadi "B milik A" atau "B A".

Contoh-contoh kalimat :

Anata no namae = Nama (milik) kamuKi no ue = Atas (nya) pohonAo no warutsu = Waltz biru

- - -

Gakkou「学校・がっこう」

Artinya "Sekolah". Sekolah di sini adalah sekolah pada umumnya, seperti SD, SMP, SMA, dsb.

17. apakah perbedaan part, season, cour, dan chapter dlm suatu film/komik ?​


Berikut ini penjelasan mengenai perbedaan part, season, cour, serta chapter yang biasa dijumpai pada film, buku atau komik:

Part atau bagian dalam film atau komik merujuk pada subdivisi utama yang membagi cerita menjadi bagian-bagian yang lebih kecil. Contohnya, sebuah film dengan tiga bagian yang berbeda dapat disebut sebagai "trilogi". Bagian-bagian tersebut sering kali memiliki alur cerita dan fokus yang berbeda.Season atau musim dalam konteks televisi atau anime merujuk pada serangkaian episode yang ditayangkan dalam periode waktu tertentu, biasanya dalam per sekian bulan atau bahkan satu tahun. Biasanya satu musim memiliki 12 hingga 24 episode, tergantung pada genre dan durasi episode.Cour atau kouru dalam anime adalah istilah yang digunakan untuk menggambarkan pembagian episode dalam satu musim. Istilah "cour"  awalnya digunakan menggambarkan rentang episode selama satu musim. Setiap kouru biasanya terdiri atas 10 - 14 episodeChapter adalah nama lain 'bab' yang biasa kita jmpai dalam komik atau buku. Chapter merujuk pada bagian-bagian besar cerita yang terpisah satu sama lain. Setiap chapter biasanya memiliki memiliki kisah dan alur cerita tersendiri meskipun tentu masih berhubungan dengan cerita utama.Pembahasan

Secara umum bisa dikatakan bahwa part, season, dan cour mengacu pada pembagian cerita pada media audiovisual. Adapun chapter, lebih cenderung mengacu pada pembagian cerita dalam bentuk media cetak seperti komik dan buku. Meski begitu, beberapa anime juga menggunakan istilah chapter untuk mengacu pada episode dalam serinya.

nihon no gakkou = sekolah jepang
nangatsu = bulan apa (kata untuk menanyakan bulan)
kara = dari
made = sampai
("kara" dan "made" sering digunakan untuk menanyakan atau menyatakan jangka waktu)
