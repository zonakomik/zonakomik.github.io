Terjemahan dan jawaban soal bahasa Jepang.

Kemarin lusa adalah hari kamis. Lusa hari apa? Asatte wa getsuyoubi desu. (Lusa adalah hari senin)Hari ini tanggal 6. Besok tanggal berapa? Ash1ta wa nanoka desu. (besok tanggal 7)3. Kapan ulang tahunmu? Watashi no tanjoubi wa ichigatsu futsuka desu. (ulang tahun saya 2 Januari)

Kata bergaris bawah dapat diganti sesuai dengan tanggal ulang tahunmu. Simak penjelasan berikut.

Pembahasan

Nama-nama hari dalam bahasa Jepang dan artinya.

Getsuyoubi = SeninKayoubi  = SelasaSuiyoubi  = RabuMokuyoubi = KamisKinyoubi = Jum’atDoyoubi = SabtuNichiyoubi = Minggu

Nama-nama tanggal dalam bahasa Jepang.

TsuitachiFutsukaMikkaYokkaItsukaMuikaNanokaYoukaKokonokaTookaJyuu ichi nichiJyuu ni nichiJyuu san nichiJyuu yokkaJyuu go nichiJyuu roku nichiIyuu sichi nichiIyuu hachi nichiIyuu ku nichiHatsukaNijyuu ichi nichiNijyuu ni nichiNijyuu san nichiNijyuu yokkaNijyuu go nichiNijyuu roku nichiNijyuu shichi nichiNijyuu hachi nichiNijyuu ku nichiSanjyuu nichiSanjyuu ichi nichi

Nama-nama bulan dalam bahasa Jepang.

Ichigatsu = JanuariNigatsu = FebruariSangatsu = MaretShigatsu = AprilGogatsu = MeiRokugatsu = JuniSichigatsu = JuliHachigatsu = AgustusKugatsu = SeptemberJyuugatsu = OktoberJyuuichigatsu = NovemberJyuunigatsu = Desember

Penjelasan butir soal.

Jika kemarin lusa adalah Kamis, maka hari ini adalah Sabtu. Lusa adalah Senin. Ototoi artinya kemarin lusa, sedangkan asatte adalah lusa.Jika hari ini adalah tanggal 6, maka besok adalah tanggal 7. Kyou artinya hari ini, sedangkan ash1ta artinya besok.Tanggal ulang tahun dapat diubah sesuai dengan tanggal lahirmu. Nama bulan ditulis terlebih dahulu kemudian nama tanggal. Perhatikan nama-nama tanggal dan bulan di atas.

35. are we Sensei no jisho desuka Terjemahkan ke dalam bahasa Indonesia​


あれは先生の辞書ですか? 「あれはせんせいのじしょですか?」 Are Wa Sensei no Jisho Desu Ka? Apakah Itu ( Jauh ) Kamus Milik Guru?

Pembahasan

Pola Kalimat:

Kata Tunjuk + は ( Wa ) + KB1 + の ( No ) + KB2 + です ( Desu ) + か ( Ka )

あれは先生の辞書ですか? 「あれはせんせいのじしょですか?」 Are Wa Sensei no Jisho Desu Ka? Apakah Itu ( Jauh ) Kamus Milik Guru?

言葉 「ことば」

• あれ ( Are ) Itu - Jauh

• は ( Wa ) Partikel

• 先生 ( Sensei ) Guru

• の ( No ) Partikel

• 辞書 ( Jisho ) Kamus

• です ( Desu ) Tidak Memiliki arti Berfungsi sebagai Penghalus Kalimat

• か ( Ka ) Partikel Tanya

_____________________________

Penjelasan

Kata Ganti Penunjuk

Berikut merupakan beberapa kata ganti Penunjuk:

これ ( Kore ) Iniそれ ( Sore ) Ituあれ ( Are ) Itu "Jauh"この ( Kono ) Benda iniその ( Sono ) Benda ituあの ( Ano ) Benda itu "Jauh"ここ ( Koko ) Siniそこ ( Soko ) Situあそこ ( Asoko ) Sana

は ( Wa ) Partikel

Wa merupakan Partikel , Partikel wa ini berfungsi sebagai penunjuk subjek dari suatu kalimat , atau bisa disebut sebagai penunjuk dari pokok kalimat, Wa ditulis menjadi ( Ha ) bukan ( Wa ) , Tetapi tetap dibaca wa

"は ( Ha )" Benar"わ ( Wa )" Salah

Fungsi partikel "Wa" : https://brainly.co.id/tugas/13512315

1.)Kore

Memiliki arti "Ini" , Digunakan untuk menunjukkan Benda yang dipegang atau didekat Pembicara , Tetapi jika lawan bicara ( pada jarak yang jauh ) tidak bisa menggunakan kata "Kore"

2.)Sore

Memiliki arti ''Itu( Jarak dekat)" Hanya digunakan pada jarak yang dekat tidak boleh pada jarak yang jauh

3.)Are

Memiliki arti ''Itu( Jarak Jauh)" Hanya digunakan pada jarak yang Jauh tidak boleh pada jarak yang dekat
